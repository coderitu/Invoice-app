# Invoice-app
An automated solution for creating invoices as per a template and converting it into PDFs which are then sent on email through outlook.
Always make sure your Outlook app is open for this application to work.

App.py is the main module.
You can add your desired Logo at the top of the generated PDF invoice.
Create an excel with the below columns and populate with your data then import the excel with all the fields as below.

| Member |	Name |	Email |	Renewal	| Invoice no.	| Pending 2025 |	Pending 2024 |	Pending 2023 |


You can preview the mail and PDF to be sent or you can bulk send for evryone in the excel at once.
